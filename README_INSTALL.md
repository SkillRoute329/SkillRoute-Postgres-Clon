# 📱 Guía de Instalación - TransForma UCOT (Android)

¡Bienvenido al futuro operativo de UCOT! El sistema TransForma ahora vive como una aplicación nativa en tu dispositivo.

## 📥 Paso 1: Descargar el Instalador (APK)

1. Ingresa al repositorio GitHub: [https://github.com/Ucot2025/TransformaFacil-2.0](https://github.com/Ucot2025/TransformaFacil-2.0)
2. Ve a la pestaña **Actions**.
3. Haz clic en el último flujo con **Check Verde** (Ej. "📱 BUILD: Triggering Android Factory").
4. Desplázate hasta abajo a la sección **Artifacts**.
5. Descarga el archivo `TransForma-App`.
6. Descomprime el archivo ZIP. Adentro encontrarás `app-debug.apk`.

> **Nota:** Envía este archivo APK a tu celular por WhatsApp, USB o Google Drive.

## 🔓 Paso 2: Instalación en Android

Al intentar abrir el archivo APK, Android te pedirá permisos de seguridad.

1. Toca el archivo para abrirlo.
2. Si aparece "Instalación bloqueada", toca en **Configuración**.
3. Activa el interruptor **"Confiar en esta fuente"** (o "Instalar apps desconocidas").
4. Vuelve atrás y toca **Instalar**.

## 🚀 Paso 3: Primer Inicio

1. Abre la app **TransForma**.
2. Ingresa con tus credenciales habituales.
   * **Usuario Maestro (Emergencia):** `SuperAdmin` / `0000` (Si aplica).
3. Concede los permisos de **Cámara** y **Ubicación** cuando la app los pida.

---

### Solución de Problemas

* **¿Pantalla blanca?** Verifica tu conexión a internet.
* **¿No deja instalar?** Asegúrate de borrar cualquier versión anterior de la app.
* **¿Aviso de Play Protect?** Toca en "Más detalles" -> "Instalar de todas formas" (Es normal en apps corporativas internas).

**Soporte Técnico:** Contactar al Área de Sistemas (Interno 404).
