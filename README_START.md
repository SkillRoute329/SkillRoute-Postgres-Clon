# Cómo iniciar TransformaFacil 2.0

Para asegurar que la aplicación se mantenga funcionando continuamente:

1.  Busque el archivo **`start-app.bat`** en esta carpeta principal.
2.  Haga doble clic sobre él.
3.  Se abrirán dos ventanas negras (Terminales):
    *   Una para el **Backend** (Servidor).
    *   Otra para el **Frontend** (Página Web).
4.  **No cierre estas ventanas** mientras quiera usar la aplicación. Minimícelas si desea.

La aplicación estará disponible en: [http://localhost:5173](http://localhost:5173)

Si necesita detener la aplicación, simplemente cierre las ventanas negras.
