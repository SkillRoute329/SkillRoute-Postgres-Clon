❓ Solución de Problemas

Problema: No puedo acceder a localhost:3005

Solución: Intenta acceder a http://localhost:3005/demo-agentes.html directamente. Si aún no funciona, el servidor puede haber caído. Reinicia el servidor con: npm run bridge en la carpeta del backend.

Problema: El navegador muestra error

Solución: Usa la página de Demo Interactiva (es HTML puro sin dependencias complejas). Limpia el caché del navegador: presiona Ctrl+Shift+Delete (Windows) o Cmd+Shift+Delete (Mac).

Problema: Los endpoints devuelven error

Solución: Verifica que el servidor backend está corriendo en puerto 3099. Abre http://localhost:3099/health - debe mostrar {"ok": true}
